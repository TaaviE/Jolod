Loosindus

Loosindus is a very simple web page for organizing Secret Santa events with your friends, family and colleagues. By just using Loosindus you have to worry a lot less. It does the annoying lifting for you.
Try now!

Getting started

It's very simple to start using Loosindus:

1.
Type in all the people
You just have to provide people's names and e-mail addresses. See exactly how.
2.
Send out invites
It takes just one button press to send out invites to people.
3.
Start the event
When everyone is ready you can generate a secret list of who gifts to who.
Try it now!
